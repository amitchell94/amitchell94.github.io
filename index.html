    Cayapas Telemedicine Android App

    An Android application to allow doctors working in the Afro-Ecuadorian and indigenous communities to submit teleconsultations for comment from specialists.

    An Android application developed in conjunction with the Cayapas Telemedicine Web Application to allow rural doctors working in Afro-Ecuadorian and indigenous communities of the upper-Cayapas river to submit teleconsultations for comment from specialists.

    Users are able to log in, fill in teleconsultation forms with patient information and photos, upload saved teleconsultations as well as download any responses received from a specialist.

    The app also features a chat function to allow users to send and receive messages about their teleconsultation to and from the specialist assigned to their teleconsultation.

    The app was designed for use in areas with poor internet access, allowing users to save teleconsultations in a local database which can then be uploaded later when the user has an internet connection.

    Developed as part of my Master's thesis at the Universidad Politécnica de Madrid.

    • Date: November 2022
    • Language: Kotlin

    Cayapas Telemedicine Web Application

    A web platform to support healthcare activities for Afro-Ecuadorian and indigenous communities around the upper-Cayapas river, Ecuador.

    The Cayapas Telemedicine Web Application was developed as part of a collaboration between TEDECO and the Amigos del Cayapas Association.

    The application supports three user roles: Coordinator, Specialist and Rural Doctor.

    Rural Doctors
    Users with the Rural Doctor role are able to write and submit teleconsultations.
    Coordinator
    Users with the Coordinator role are able to assign teleconsultations to specialists as well as create and approve new user accounts.
    Specialists
    Users with the Specialist role are able to write and send responses to the teleconsultations assigned to them.

    After a specialist responds to a teleconsultation, a chat feature allows the Rural doctor and Specialist users to send and receive messages to discuss the patient's case.

    Technologies used: Phoenix, Ecto, Guardian, JWT, Bootstrap, PostgreSQL and a RESTful API to allow access for a mobile applicaton.

    Developed as part of my Master's thesis at the Universidad Politécnica de Madrid.

    • Date: November 2022
    • Language: Elixir
